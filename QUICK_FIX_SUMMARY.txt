SHARING FEATURE - QUICK FIX SUMMARY

ROOT CAUSE:
-----------
1. Tried to store EMAIL in UUID columns → Type mismatch errors
2. RLS policies querying same table → Infinite recursion
3. Complex policy dependencies → Can't alter columns

THE FIX:
--------
1. Keep UUID columns (proper database design)
2. Add email→UUID lookup function
3. Simplify all RLS policies (no recursion)
4. Update Flutter to use lookup
