│ Scalability      │ O(n) requests       │ O(1) connection                 │

╔══════════════════════════════════════════════════════════════════════════════╗
║                         CONNECTION LIFECYCLE                                 ║
╚══════════════════════════════════════════════════════════════════════════════╝

App Start
   │
   ├─> Check connectivity
   │      │
   │      ├─> Online? ──> Start RealtimeService
   │      │                  │
   │      │                  ├─> Subscribe to items
   │      │                  ├─> Subscribe to blocks
   │      │                  └─> Subscribe to notifications
   │      │
   │      └─> Offline? ──> Wait for connectivity
   │
Network Change
   │
   ├─> Went Online? ──> Start RealtimeService + Sync
   │
   └─> Went Offline? ──> Stop RealtimeService
   
App Close
   │
   └─> Stop RealtimeService
          │
          ├─> Unsubscribe from items
          ├─> Unsubscribe from blocks
          └─> Unsubscribe from notifications
